# Hegemony reference materials

These PDFs are the official rulebooks for *Hegemony: Lead Your Class to Victory* and the *Crisis & Control* expansion, plus the publisher's classroom guide. They live in this repo so the tracker has its source of truth at hand — they are **not** redistributed, only mirrored locally for the project.

If a newer rulebook is posted by Hegemonic Project Games, replace the PDF in place and bump the version number in this file.
